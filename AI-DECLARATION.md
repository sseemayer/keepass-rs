## Notes

We align with [KeePassXC's philosophy on code generated by generative AI](https://keepassxc.org/blog/2025-11-09-about-keepassxcs-code-quality-control/).

The crate was fully designed by humans. 

Most (>90%) implementation, unit tests, and documentation was written by humans with autocompletion provided by rust-analyzer and LLM-based tab-completion. Small portions of the code were generated by AI agents guided by humans.

All code was [reviewed by humans](.github/CODEOWNERS). Since 2023, all code requires a review by a maintainer other than the pull request author before it can be merged into the main branch.
